# KOSIS(국가통계포털) 데이터 수집 및 분석 프로젝트

이 폴더는 통계청 KOSIS Open API를 활용하여 국내 통계 데이터를 수집하고 분석하기 위해 구성된 개별 프로젝트 폴더입니다.

## 📁 폴더 구조

- **docs/**: 통계청 API 가이드 및 관련 문서
- **src/**: 데이터 수집 및 분석 소스코드
- **data/**: API를 통해 수집된 원본 및 가공 데이터 (CSV, Excel 등)
- **report/**: 데이터 분석 결과 보고서 및 시각화 결과물
- **images/**: 보고서 및 분석에 활용되는 시각화 이미지

## 🔑 KOSIS Open API 사용 방법

1. **KOSIS 공유서비스 회원가입 및 로그인**
   - [KOSIS 공유서비스 홈페이지](https://kosis.kr/openapi/)에 접속하여 회원가입을 진행합니다.

2. **인증키(API Key) 신청**
   - 로그인 후 [마이페이지] -> [인증키 신청] 메뉴에서 API 사용을 위한 인증키를 발급받습니다.

3. **통계표 ID 확인**
   - 수집하려는 통계표(예: 주민등록인구현황 등)의 상세 화면에서 통계표 ID(예: `DT_1B040M3` 등)를 확인합니다.

4. **API 호출**
   - `src/scrape_kosis.py` 코드에 발급받은 API 인증키와 수집하고자 하는 통계표 ID를 입력한 후 실행하여 데이터를 가져옵니다.
